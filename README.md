# 미국 주식 알리미

## 개요

- 미국 주식 Ticker를 등록하고 관리할 수 있다. 
- 인디케이터를 등록해두면, 티커에 해당하는 인디케이터를 조합하여, 매수/매도 시점을 Slack으로 알림을 보낸다.

## 사용기술

- 서버:
    - Python FastAPI
    - Python SQLite3
    - SQLAlchemy
    - Pandas
    - yfinance  
- 클라이언트:
    - TailwindCSS 4
    - SvelteKit
    - Svelte Components
    - Svelte Icons
    - Svelte Store
    - Svelte Action
    - Svelte Transition
    - Svelte Animation
    - Svelte Scroll
